

This project is a simple FAQ Chatbot developed using Python, NLTK, and Machine Learning techniques. The chatbot is designed to answer user questions by matching them with the most relevant frequently asked questions stored in the system.

The project uses Natural Language Processing (NLP) techniques such as tokenization, stopword removal, and text preprocessing to clean and process user input. TF-IDF vectorization and cosine similarity are used to compare the user’s question with the stored FAQs and identify the best matching answer.

A simple chatbot interface is provided for user interaction, where users can type questions and receive instant responses. This project demonstrates the practical implementation of NLP, text similarity, and chatbot development using Python.
